# 🍽️ Food Demand Prediction AI

This project predicts food demand using Machine Learning and provides business recommendations.

## 🚀 Features
- Demand prediction using XGBoost
- Weather + weekend awareness
- Business recommendation system
- Streamlit UI

## 🧠 Tech Stack
- Python
- Scikit-learn / XGBoost
- Streamlit

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
